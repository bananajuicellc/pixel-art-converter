# Document Object (v3)

The root object in `document.json`.

| Property | Type | Description |
| :--- | :--- | :--- |
| `sprites` | Array | A collection of [Sprite](sprite.md) objects. |
| `palette` | Object | Color palette definition. |
| `gridSettings` | Object | Canvas grid configuration. |
| `animationSpeed` | Number | Frames per second. |
| `primarySpriteIdentifier` | String | The ID of the main sprite. |
