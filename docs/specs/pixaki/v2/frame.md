# Frame Object (v2)

A single snapshot in a symbol sequence.

| Key | Type | Description |
| :--- | :--- | :--- |
| `Duration` | Integer | Frame hold duration (multiplied by inverse animation speed). |
| `Layers` | Array | A collection of [Layer](layer.md) objects. |
| `SelectedLayerIndex` | Integer | Currently active layer. |
| `Hidden` | Boolean | (Optional) Whether the frame is hidden. |
