# Symbol Object (v2)

A Symbol contains a sequence of frames.

| Key | Type | Description |
| :--- | :--- | :--- |
| `Name` | String | Display name of the symbol (e.g., "Foreground"). |
| `Frames` | Array | A collection of [Frame](frame.md) objects. |
| `SelectedFrameIndex` | Integer | Currently active frame within the symbol. |
