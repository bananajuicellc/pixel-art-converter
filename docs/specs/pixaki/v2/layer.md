# Layer Object (v2)

References an image file within the Pixaki package.

| Key | Type | Description |
| :--- | :--- | :--- |
| `ImageFilename` | String | The filename of the PNG file (e.g., `Layer[GUID].png`). |
| `Alpha` | Number | Opacity (0.0 to 1.0). |
| `Visible` | Boolean | Visibility status. |
| `Reference` | Boolean | Whether this is a reference layer. |
| `Size` | Dictionary | `{ Width, Height }` (floats). |
